### 第1章 MySQL体系结构和存储引擎

#### 1.1 定义数据库和实例

**MySQL数据库实例在系统上的表现就是一个进程**

理解数据库和数据库实例：

**数据库**是文件的集合，依照某种数据模型组织起来并存放于二级存储器中的数据集合。

二级存储器，指硬盘，固态硬盘。

一级存储器是主存。



**数据库实例**是位于用户和操作系统之间的一层数据管理软件，用户对数据库数据的任何操作都是在数据库实例下进行的。

应用程序只有通过数据库实例才能和数据库打交道。

所以，MySQL其实是数据库实例？（思考）

好吧，其实在实际中不会咬文嚼字，但用户一般都简单的把MySQL理解成数据库依旧有失偏颇。
